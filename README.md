it's a site.

## Requirements
* Laptop running Linux, Unix, or Mac OS X
* Ruby and Git Installed
* GitHub Account
* Text Editor (TextMate, Sublime, VIM)

## Setup Jekyll

#### Create a new repo
```
yourusername.github.io
```

#### Clone the repo
```
git clone git@github.com:yourusername/yourusername.github.io.git
```

#### Change directories
```
cd yourusername.github.io
```

#### Install Jekyll
```
gem install jekyll
```

#### Create new blog
```
jekyll new blog
```

#### Move directory
```
mv blog/* ./
```

#### Delete blog folder
```
rm -rf blog
```

#### Start the server
```
jekyll serve -w
```

#### Visit
```
localhost:4000
```
